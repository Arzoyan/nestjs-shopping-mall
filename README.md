## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ git clone https://github.com/Arzoyan/nestjs-shopping-mall.git
```

navigate to root directory

```bash
$ cd nestjs-shopping-mall
```

Install project

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev
```
